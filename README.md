# AI Restaurant Booking System

## 🚀 Overview
A smart restaurant reservation system built with Spring Boot, featuring intelligent table allocation, analytics, and predictive demand scoring.

---

## 🧠 Features

### 🍽 Booking Engine
- Smart table allocation based on capacity
- Conflict-free reservation system
- Real-time availability checks

### 📊 Analytics Module
- Peak hour analysis
- Average party size tracking
- Booking distribution insights

### 🤖 Predictive Intelligence
- Demand scoring per hour
- Busy time prediction
- Smart booking suggestions

---

## 🏗 Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven

---

## 🧱 Architecture
- Controller → Service → Repository pattern
- RESTful APIs
- Exception handling with global handler

---

## 📂 Database
- Tables: reservations, restaurant_tables
- Seed data included for full-day simulation

---

## 📈 Future Improvements
- Machine learning-based demand prediction
- Admin dashboard UI
- Real-time booking notifications
